Git e Github

*Comandos:

** pull: (puxar) fazer download dos arquivos do repositório que foram modificados
** push: (empurrar) fazer o upload para o repositório 

** git clone: faz uma cópia do repositório para a sua IDE
*** clonar branch: fazer uma ramificação para realizar alterações sem mudar o original

** git branch: manipular as ramificações 
*** 'git branch <noe-da-branch>'

** Criar a branch e já enviar ao repositório ao mesmo tempo
	'git push -u <local-remoto><nome-da-branch>'

** deletar branch:
*** 'git branch -d'

*** 'git merge <nome-da-branch-com-o-recurso>'

** git status: confere se precisamos fazer pull ou push
